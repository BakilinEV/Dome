Регистр сведений "Высоковостребованный неснижаемый запас"

Регистр предназначен для хранения сведений о количественном минимальном пороге номенклатуры (запчасти) для основного средства организации.

Регистр содержит несколько форм:

  • Форма Отчет - формирует отчет по неснижаемому запасу (Сервис ® Дополнительные отчеты и обработки ® Дополнительные внешние отчеты)
  • Форма Загрузки данных - для загрузки данных в базу (через форму Отчет - по кнопке "Загрузить")

    • Примечание по отчету:

    1. При минимальном остатке позиций, строка выделяется красным цветом для оперативного заказа поставщику.

    2. Расход показывает с даты последнего закупа

    3. Остатки на складе Юралс (склад ПТК) отображаются на текущую дату
